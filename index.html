PROPRESENTER BUILD GUIDE

📐 The Core Principle

Every slide has two distinct layers:

Layer 1 — Background Art: Fills the whole screen. Giant atmospheric type, textures, gradients. This is DECORATION — it's OK if the preacher blocks it.

Layer 2 — Content: Always sits in the top 38-40% of the screen. Scripture, points, quotes. ALWAYS above the preacher's head.

🏗️ Building in ProPresenter

1. Import background PNG (download from here)
2. Add text box for content — set it to top 38% of slide only — lock it there
3. Add left accent bar (thin rectangle, accent colour, full height of content area)
4. Save as a Master Slide in your theme
5. Each Sunday: duplicate master, update text only

Total weekly build time: under 10 minutes.

🔥 Background Art Strategy

The giant "MAIN" / atmospheric text in the background is what makes it look like series artwork. Here's how to get it in ProPresenter:

• Download the PNG backgrounds from this tool
• OR in ProPresenter, add a text box with giant Anton font text, set to 5% opacity, position it in the lower 60%
• Size it so it bleeds off the edges — that's the editorial look
• It will look incredible on camera even when the preacher stands in front

📺 Stream Considerations

With the preacher in front of the lower zone, the camera actually helps you — it frames the preacher against the atmospheric background art, which looks intentional and cinematic.

The top zone text is always fully visible on stream because it's above the preacher. This is exactly how Elevation Church does it — the slide looks designed even mid-sermon.

🎨 Changing Series

When you move to a new series:

1. Change the background style (9 options here)
2. Change the accent colour
3. Pick a title layout (Left / Center / Wide)
4. Update the series watermark text (bottom right)
5. Everything else stays the same

New series = new vibe in literally 5 minutes.
